# patterns

> :notebook_with_decorative_cover: Simplest implementation of design patterns in JavaScript.

## Examples

List of design patterns which they have example. 

```
$ npm run model-view-controller
$ npm run observer
$ npm run strategy
$ npm run factory-method
$ npm run singleton
$ npm run mediator
```
